# โปรเจกต์ POS เป็นการฝึกทำโปรเจกต์

สิ่งที่รับจากโปรเจกต์

1. เข้าใจภาพรวมเพราะหลักๆคือการใช้ API และ Global State
2. เจอปัญหาระหว่างทำโปรเจกต์ เพราะไม่ได้วางแผนก่อนทำ ตอนเขียนโค้ดเลยทำให้หลงทาง
3. ได้ใช้ AI ช่วยอธิบาย หรือ Guide Line ในการเขียนโค้ด เพราะเรารู้ว่าไปต่อยังไง แต่ลืมวิธีเขียน

## Tech Stack

| รายการ | คำอธิบาย |
| ------ | ----------- |
| ReactJS   | ใช้เป็น frontend ในการดึง API และใช้ useState, useEffect, Global State, Navigate, Component ในการจัดการหน้าเว็บ. |
| NodeJS | ใช้เป็น Enviroment ในการติดตั้ง Package และ Run Server. |
| ExpressJS    | ใช้ในการเขียน API Backend เพื่อให้ Frontend ดึงไปใช้งาน. |
| MySQL    | ใช้ในเก็บข้อมูล. |

## Tools 

| รายการ | คำอธิบาย |
| ------ | ----------- |
| VSCode   | เป็น IDE ในการเขียนโค้ด เพราะมี Extenstion ทำเขียนโค้ดได้เร็วและไม่เหนื่อย |
| Postman | ใช้ทดสอบ API Endpoint ว่าส่งค่าข้อมูลได้ตามที่ต้องการ  . |
